# Validate Command

Run comprehensive validation checks on the codebase.

## Tasks

1. Run `npm run check` - Full validation suite
2. Report any errors found
3. If errors exist, offer to fix them
4. Run validation scripts:
   - Import validation
   - Export validation
   - Secret scanning
   - Code quality checks
5. Summarize results

## Success Criteria

- All linting passes
- All type checks pass
- All tests pass
- All validation scripts pass
- No secrets detected
- Code quality score is high
