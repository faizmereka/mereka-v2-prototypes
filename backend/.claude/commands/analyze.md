# Analyze Command

Analyze project status, identify issues, and provide recommendations.

## Tasks

1. **Code Quality Analysis**:
   - Run `npm run check:fast`
   - Identify linting issues
   - Check TypeScript errors
   - Review import organization

2. **Test Coverage Analysis**:
   - Run coverage report
   - Identify gaps in testing
   - List untested modules

3. **Security Analysis**:
   - Run secret scanning
   - Check for vulnerable dependencies
   - Review authentication implementation

4. **Performance Review**:
   - Identify missing database indexes
   - Review query patterns
   - Check for N+1 queries

5. **Documentation Review**:
   - Check for missing API documentation
   - Verify README is up to date
   - Ensure all modules are documented

6. **Provide Recommendations**:
   - Prioritized list of improvements
   - Quick wins
   - Long-term enhancements

## Output

Comprehensive analysis report with:

- Current health score
- Critical issues
- Warnings
- Recommendations
- Action items
