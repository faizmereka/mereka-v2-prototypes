# Test Command

Run tests for the project and provide coverage analysis.

## Tasks

1. Run `npm run test:coverage` to get current coverage
2. Analyze coverage report:
   - Identify files with <80% coverage
   - Find untested functions
   - Detect missing test files
3. Offer to create missing tests if coverage is low
4. Run integration tests
5. Report test results with details

## Success Criteria

- All tests pass
- Coverage ≥80% for lines, functions, branches, statements
- No test failures
- Clear summary of test results
