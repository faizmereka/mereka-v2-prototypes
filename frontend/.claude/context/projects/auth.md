# Auth Project Context

## Project: auth (auth.mereka.io)

Authentication app for login, registration, and password reset.

## Location
`projects/auth/`

## Features
- Login
- Registration
- Password reset
- Email verification
- Social login (Google, Facebook, Apple)

## Commands
```bash
# Serve
ng serve auth --port 4201

# Build
ng build auth --configuration=production
```
